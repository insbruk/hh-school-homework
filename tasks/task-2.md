# Сохранять выбранный фильтр и поисковый запрос

## Описание
1. Введенное значение в строку поиска должно сохраняться после обновления страницы
2. Выбранное значения фильтра должно сохраняться после обновления страницы

## Дополнительно
1) В LocalStorage использовать ключ `searchTitle` для хранения введенного значения в поисковую строку
2) В LocalStorage использовать ключ `filterStatus` для хранения выбранного значения фильтра
