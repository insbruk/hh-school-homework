# Добавить login

## Описание
1. По клику на "Войти" добавить запрос `POST /login` (используя `fetch`)
2. В теле запроса нужно передать username/password в формате JSON (как в примере ниже)
3. username/password можно передавать любые
4. Если запрос успешный в ответе вернется JSON с токеном
5. Полученный токен нужно сохранить в localStorage используя ключ `accessToken`

## Если все правильно сделано
* Кнопка "Войти" переименуется в "Выйти"
* Отрендерится поисковая строка, фильтр, кнопка "Искать", заголовок "Список задач"

### Запрос
> POST http://localhost:5000/login HTTP/1.1
> 
> Content-Type: application/json

> `{ "username": "ivan", "password": "123"}`

### Ответ
>HTTP/1.1 200 OK
> 
> `{ "accessToken": "..." }`
