# Добавить logout

## Описание
1. По клику на кнопку "Выйти" добавить запрос `POST /logout` (используя `fetch`)
2. Если запрос выполнился успешно - сбросить `accessToken` в localStorage
3. API доступно только с `accessToken`

## Если все правильно сделано
* Кнопка "Выйти" переименуется в "Войти", локальное хранилище очистится
* Поисковая строка, фильтр и список задач скроется
* Logout сбросит токен на сервере

### Запрос
> `POST /logout HTTP/1.1`
> 
> Host: localhost:5000
> 
> Authorization: Bearer {{accessToken}}

### Ответ
> `HTTP/1.1 200 OK`
